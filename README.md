## âš™ï¸ How It Works

1. The extension injects a custom script into specific websites.  
2. It hides or modifies distracting UI elements based on your settings.  
3. All features are executed **locally**, meaning no data is ever sent anywhere.  
4. You can toggle features directly from the popup.
